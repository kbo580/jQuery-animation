アニメーション

show( )/hide( )

show( );/hide( );
show( )は表示、hide( )は非表示にする
$('セレクター').show(スピード);
$('セレクター').hide(スピード);
( )内には「"fast"」,「"slow"」のいずれか、またはミリ秒単位で(1000ミリ=1秒)スピードを入力する。ミリ秒で指定する場合は” ”で囲わなくても良い。何も記入しないと即座に変化する
$('#btn').click(function(){
 $('#box').hide("fast");
  });
$('#btn').click(function(){
  $('#box').show("slow");
 });
表示の方は「"slow"」非表示の方は「"fast"」を設定した
toggle( );
一つのコマンドで表示、非表示をする
$("セレクター").toggle(スピード);
$('#btn').click(function(){
  $('#box').toggle(500);
  });
クリックイベントの場合だとボタンを連打するとその数だけアニメーションが実行されてしまう
:not(:animated)と組み合わせることで解決できる。これを書くとアニメーションが処理中ではない要素を選択できるため
$('#btn').click(function(){
 $('#box:not(:animated)').toggle(500);
 });

slideDown( )/slideUp( )

slideDown( )/slideUp(v)
slideDown( )はセレクターで指定した要素をスライドダウンで表示、/slideUp( )はスライドアップで非表示をすることができる
$('セレクター').slideDown(スピード);
$('セレクター').slideUp(スピード);
$('#btn').click(function(){
  $('#box').slideDown('slow');
 });
$('#btn').click(function(){
  $('#box').slideUp('fast');
 });
slideToggle( )
slideDown( )/slideUp( )にもtoggleが使える
$('セレクター').slideToggle(スピード);
$('#btn').click(function(){
 $('#box:not(:animated)').slideToggle("slow");
 });

fadeIn( )/fadeOut( )

fadeIn( )/fadeOut( )
フェードインしながら要素を表示、非表示する
$('セレクター').fadeIn(スピード);
$('セレクター').fadeOut(スピード);
$('#btn').click(function(){
  $('#box').fadeIn("slow");
 });
$('#btn').click(function(){
  $('#box').fadeOut("fast");
 });
fadeTo( )
セレクターで指定した要素の透明度を設定まで徐々に変更する
$('セレクター').fadeTo("スピード",透明度);
$('#btn').click(function(){
 $('#box').fadeTo("slow",0.5);
 });
fadeToggle( )
fadeIn( )/fadeOut( )にもtoggleが使える
$('セレクター').fadeToggle(スピード);
$('#btn').click(function(){
  $('#box':not(:animated)').fadeToggle("slow");
 });
fadeOut("slow")とfadeTo("slow",0)は似ているが結果に違いがある。fadeOut("slow")の方はスペースを埋めてしまうのに対して、fadeTo("slow",0)は透明度が０になっているだけで実際はそこに存在している。使う時はどちらの結果を得たいかで使い分ける。

animate( )

animate( )
数値を扱うプロパティのみとなる(bakground-colorやdisplay)
$("セレクター").animate({
  "cssセレクタ": "css数値"
 });
これだけで最低限書ける
$('#btn').click(function( ){
 $('#box').animate({
  "margin-left": "500px",
  });
  });
{ }の後ろをカンマで区切って時間をミリ秒単位で指定する事でアニメーションのスピードを変えられる
$("セレクター").animate({
  "cssセレクタ": "css数値"
 },アニメーション時間);
上より少し遅い
$('#btn').click(function( ){
 $('#box').animate({
  "margin-left": "500px",
  },1000);
  });
{ }の後ろをカンマで区切って' 'の中に「linear」か「swing」を書く事でアニメーションの緩急を指定できる。この緩急をeasingといいデフォルトでは２つしか使えないがプラグインを使えば他のも使えるようになる。
$("セレクター").animate({
  "cssセレクタ": "css数値"
 },アニメーション時間,'緩急');
一応緩急がついているwww
$('#btn').click(function( ){
 $('#box').animate({
  "margin-left": "500px",
  },1000,'swing');
  });
長ったらしいけどコールバック関数を使う事で複雑なこともできる
$('#btn').click(function( ){
 $('#box').animate({
  "margin-left": "500px",
  "width": "50px",
  "height": "50px"
 },1000,'swing',function( ){
   $(this).css("background-color","red")
  .animate({
 "width":"200px",
 "height":"200px",
 },1000,function( ){
  $(this).fadeOut('slow');
   });
  });
  });
